💡 Getting Your Price List from EB Horsman
  1. Log into your EB Horsman account at ebhorsman.com
  2. Go to My Account → Price List or ask your rep to email it
  3. Export as Excel (.xlsx) or CSV

📱 Share with Co-Owner
Since this app is a single HTML file, here are your options for sharing it with your co-owner so they can use it on their phone or computer.
Option 1 — Share the file directly (simplest)
  1. Click ⬇ Export to save a backup JSON of your data
  2. Send the PackOps_Dashboard.html file to your co-owner via email, AirDrop, or Google Drive
  3. They open it in any browser — works on iPhone/Android in Chrome or Safari
  4. They click ⬆ Import and load your JSON to get the same data
  5. Each person has their own copy — export/import to sync when needed
Option 2 — Host on GitHub Pages (free, always online)
  1. Create a free account at github.com
  2. Create a new repository, upload your HTML file as index.html
  3. Go to Settings → Pages → Source: main branch
  4. GitHub gives you a URL like yourname.github.io/packops
  5. Share that URL — works on any device, any browser
  6. Use Export/Import JSON to keep data in sync between devices
💡 This is the recommended path. Free, fast, and your co-owner bookmarks the URL on their phone like an app.
Option 3 — Real-time sync (future)
When you connect QuickBooks and Google APIs, we can move the data to a proper backend (Supabase — free tier) so both of you always see the same data in real time without importing. That's the end goal.
